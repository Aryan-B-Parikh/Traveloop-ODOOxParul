# Traveloop Frontend

Modern travel planning UI built with React and Vite.

Run locally:

```bash
npm install
npm run start
```

Open `http://localhost:5173`.
 
## Development (start both services)

Use the `dev:all` script to start both the backend and frontend concurrently from the repository root:

```bash
npm install
npm run dev:all
```

This runs the backend dev server and the Vite frontend together.
